# Chatwoot

Chatwoot is an open-source customer support and omni-channel messaging platform that provides REST APIs for managing conversations, contacts, agents, teams, labels, and integrating customer communication workflows. It serves as an open-source alternative to Intercom, Zendesk, and Salesforce Service Cloud.

## APIs

This repository catalogs three Chatwoot API surfaces:

- **Application API** — Agent and admin interactions: conversations, contacts, inboxes, labels, teams, webhooks, and reports. Authenticated via user access tokens.
- **Client API** — Custom messaging interfaces for end users. Authenticated via inbox and contact identifiers.
- **Platform API** — Super admin management of self-hosted installations, accounts, and users. Self-hosted only.
